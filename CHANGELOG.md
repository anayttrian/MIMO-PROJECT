## [0.1.0] - 2026-05-15

### Added
- **Stock Analysis Agent**
  - Real-time price fetching from Stockbit API
  - News aggregation from Kompas, CNBC Indonesia, CNN Indonesia
  - Technical analysis (RSI, MACD, support/resistance levels)
  - Fundamental analysis with catalyst identification
  - Trading setup generation with entry/exit points
  - Risk/reward ratio calculation
  - Markdown report generation

- **Content Generation Pipeline**
  - Automated social media thread creation
  - Multi-platform support (Twitter, Telegram)
  - Tone customization (aggressive/hype, educational, minimalist)
  - Emoji optimization for engagement
  - Indonesian language support
  - Multiple content variants for A/B testing

- **Multi-Agent System**
  - Master-worker agent architecture
  - Parallel task execution via delegation
  - Context-aware agent communication
  - Result aggregation and synthesis
  - Error handling with retry logic

- **Email Verification Agent**
  - Email format validation (regex)
  - DNS MX record lookup
  - SMTP mailbox verification
  - Provider detection
  - Stealth mode (no alert triggering)
  - Amazon account registration check

- **Infrastructure**
  - Hermes Agent framework integration
  - Claude Sonnet 4 as primary LLM
  - Environment configuration system
  - Logging and monitoring
  - Error handling and recovery
